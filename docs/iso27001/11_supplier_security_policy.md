# Supplier Security Policy
**Organization:** BlackMess Research
**Version:** 1.0
**Date:** April 2026

## 1. Approved Suppliers

| Supplier | Service | Security Assessment |
|----------|---------|---------------------|
| Railway | Backend hosting | SOC 2 Type II |
| Vercel | Frontend hosting | SOC 2 Type II |
| GitHub | Source control | SOC 2 Type II |
| Anthropic | AI API | Enterprise security |

## 2. Requirements
- All suppliers must maintain SOC 2 or equivalent
- Data processing agreements in place
- Encryption in transit mandatory (TLS 1.3)
- Incident notification within 72 hours

## 3. Review
Annual supplier security review conducted.

# Information Security Management System (ISMS) Policy
**Organization:** BlackMess Research
**Owner:** Akbar Ramadhan
**Version:** 1.0
**Date:** April 2026

## 1. Purpose
This policy establishes the framework for information security management at BlackMess Research, ensuring confidentiality, integrity, and availability of all information assets.

## 2. Scope
This ISMS applies to all systems, data, and processes within BlackMess platform including:
- Django backend (Railway)
- React frontend (Vercel)
- PostgreSQL database
- Redis cache
- IPFS storage

## 3. Information Security Objectives
- Maintain 99.99% uptime for critical systems
- Zero unauthorized access to encrypted data
- Full compliance with GDPR, OJK/BI, and ISO 27001
- Post-quantum cryptography implementation (FIPS 203/204)
